# Contributing to TmuxWeb

Thank you for your interest in TmuxWeb! We welcome feedback, bug reports, and feature suggestions through GitHub Issues.

## How to Contribute

### Reporting Bugs

1. Search [existing issues](https://github.com/includewudi/opencode-tmuxweb/issues) to avoid duplicates
2. Open a new issue with the **Bug Report** label
3. Include:
   - Steps to reproduce
   - Expected vs actual behavior
   - Environment info (OS, Node.js version, browser, tmux version)
   - Screenshots or logs if applicable

### Suggesting Features

1. Open a new issue with the **Feature Request** label
2. Describe the use case and expected behavior
3. Explain why this would be useful

### Asking Questions

Feel free to open an issue with the **Question** label for usage questions or help with setup.

## Code Contributions

**We are not accepting pull requests at this time.** The project is maintained by a single developer and is not yet ready for external code contributions.

If you have a code-level suggestion, please describe it in an issue instead — we'd love to hear your ideas.

## Code of Conduct

Be respectful and constructive. We're all here to make great tools.
